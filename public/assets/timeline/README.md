# Timeline Image Assets

Timeline images must be real images from, or directly documenting, the event
they are attached to. Do not use generic symbolic images, generated images, or
unrelated stock imagery here.

The website references the filenames below. If a file is missing, the timeline
card hides the image area until the real asset is added. Images are displayed in
black and white by CSS, so source files may be colour.

Recommended format:
- `.jpg`
- 16:9 crop
- 1200-1600 px wide
- compressed for web
